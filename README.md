# Knock-Knock Joke Generator

This just prints out a full knock-knock joke using `System.out.println()` statements. It's a basic exercise in Java string output.

## How to Run

javac Knock_Knock_Joke.java

java Knock_Knock_Joke

## Topics Covered
- Sequential Output  
- Basic Java Syntax  
- String Printing
